# Tilføj fællesmailkonto
---

![](faellesmail1.png)
```js
Åben outlook og klik på 'Filer'
```

![](faellesmail2.png)
```js
Vælg derefter 'Tilføj konto'
```

![](faellesmail3.png)
```js

```

```js
Indtast nu din egen mailadresse og login

```
